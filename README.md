# 🪟 Curtain Store Admin Panel & Frontend

This project is a **Curtain Store Web Application** that includes both a **user-facing frontend** and an **admin panel** in a single repository.

## 🚀 Tech Stack

* ⚛️ React (Frontend)
* 🗂 Zustand (State Management)
* 💻 JavaScript (ES6+)
* 🎨 CSS / Tailwind (if used — edit if needed)

---

## 📦 Features

### 🛍 Frontend (User Side)

* View curtain products
* Responsive product cards
* Product detail page
* Pagination system
* Clean and modern UI

### 🛠 Admin Panel

* Add new products
* Edit existing products
* Delete products
* Manage all models in one place
* Real-time UI updates with Zustand

## 🔄 State Management

This project uses **Zustand** for global state management:

* Product list stored globally
* Instant updates without Redux complexity
* Simple and scalable structure
